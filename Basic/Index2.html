Seminar 4

Pentru construirea acestei pagini web s-a folosit un layout pe trei coloane. Pagina web contine 4 sectiuni: logo, meniu, continut si subsol .

Acest seminar isi propune familiarizarea cu elementele de structura a HTML-ului si modul in care se produce arborele DOM
Se vor discuta aspecte referitoare la tagurile de baza a unei paginilor web:
  • HTML
  • HEAD
  • TITLE
  • BODY
O alta problema abordata in acest seminar este cea a geometriei paginilor web. Prin geometrie intelegem asezarea vizuala a datelor in cadrul unei pagini web. Se va insista pe structurarea informatiei folosind tabelele. Mentionam ca tabelele sunt concepte care trebuie intelese in doua moduri diferite in functie de motivul utilizarii lor. Astfel, vom discuta atat de tabele ca layout cat si ca structuri de date. In final se va analiza codul HTML ca structura de date arborescenta, ce urmeaza a fi folosita ca input pentru CSS si JavaScript.
